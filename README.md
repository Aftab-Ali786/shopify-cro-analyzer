# Shopify CRO Analyzer

An AI-powered Shopify Conversion Rate Optimization (CRO) auditing platform.

## Features

- Shopify Store Detection
- Homepage Crawler
- Product Extraction
- Navigation Analysis
- Trust Signal Detection
- Screenshot Capture
- CRO Heatmap
- Rule-Based CRO Analysis
- AI Recommendations
- Opportunity Prioritization
- Executive Dashboard

## Tech Stack

Frontend
- React
- TypeScript
- TailwindCSS
- Vite

Backend
- Node.js
- Express
- Puppeteer
- Cheerio

AI
- Google Gemini
